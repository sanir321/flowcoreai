Compliance Layer Design

Section 1: Architecture & Data Flow

Centralized Dispatch Strategy

To ensure consistency across the platform, we will move all WhatsApp compliance logic into the Agent Orchestrator. This ensures that whether a message is triggered by a webhook or a manual action, it follows the same safety rules.

New Logic Flow
  1. Start: Orchestrator receives request.
  2. Check: Is it a WhatsApp channel?
  3. Verify: Is last_customer_message_at < 24 hours ago?
    • If NO: Log "WA_WINDOW_EXPIRED" and Abort.
  4. Process: Run AI to generate response.
  5. Dispatch Cycle: For each message part:
    • Send presence: composing to GoWA.
    • Wait (length * 35ms).
    • Send message to GoWA.
    • Send presence: paused to GoWA.
